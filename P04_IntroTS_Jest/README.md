## Notas
1. Execute test:
```bash
npm run test src/...
```

2. Execute ts-node:
```bash
ts-node src/... <arguments>
```

3. Create documentation:
```bash
npx typedoc src/...
```

4. Do not add "type":"module" to the package.json
