If the server hosting the resource does not include the required Access-Control-Allow-Origin header in its response, you may not be able to access the resource directly from your web page using JavaScript's fetch API due to the same-origin policy and CORS restrictions.

One possible solution is to use a server-side script to fetch the JSON data from the remote URL and send it to your web page as a response. This way, your web page is making a same-origin request to your server, which can then make the cross-origin request to the remote URL.
